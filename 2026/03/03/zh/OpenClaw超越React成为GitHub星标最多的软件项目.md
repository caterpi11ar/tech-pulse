# OpenClaw超越React，成为GitHub星标最多的软件项目

自主智能代理平台OpenClaw在上线不到四个月后突破25万颗GitHub星标，超越长期霸榜的React前端框架，成为GitHub历史上星标最多的非聚合类软件项目。

![OpenClaw GitHub](https://opengraph.githubassets.com/e617fbb1b08448ad6246dbaea1bac2adebef1b7f1fb7208968cd68b2080fd70d/openclaw/openclaw)

## 事件经过

根据GitHub星标历史追踪网站Star History发布的分析文章，OpenClaw在2026年3月1日前后正式突破25万颗星标，超过了React此前保持多年的纪录，成为GitHub平台上星标最多的非聚合类软件项目（即排除"awesome list"等资源汇总型仓库）。

仅在一周前，Star History还曾撰文报道OpenClaw超越Linux内核，跃升至GitHub全时总榜第14名，并预言道"以这个速度，React的领先地位恐怕撑不了多久"——事实证明，这一预言很快成真。

OpenClaw从零到GitHub第一仅用了不到四个月，增长曲线几乎没有放缓的迹象。

## 背景与意义

这一里程碑反映了AI智能体（Agentic AI）领域的爆发式增长趋势。React此前的GitHub星标纪录建立在前端开发生态的长期积累之上，代表了数以百万计开发者的认可与使用。OpenClaw能在极短时间内超越这一纪录，说明开发者社区对AI驱动的自主代理工具抱有极高热情。

从更宏观的视角来看，GitHub星标数量虽然并不等同于生产使用量，但它是衡量开发者关注度与项目热度的重要指标，也反映了AI工具化正在成为当前技术栈演进的核心叙事。

值得一提的是，OpenClaw v2026.3.2版本也在近期发布，此次更新重点扩展了SecretRef的安全凭证覆盖范围（共覆盖64个目标），进一步完善了企业级安全特性。
