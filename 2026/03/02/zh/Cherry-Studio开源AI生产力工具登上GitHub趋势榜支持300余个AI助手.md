# Cherry Studio 开源 AI 生产力工具登上 GitHub 趋势榜，支持 300 余个 AI 助手

Cherry Studio 是一款开源 AI 生产力工作台，集智能对话、自主智能体和 300 余个预置 AI 助手于一身，支持统一接入各大前沿大语言模型，近日登上 GitHub 趋势榜单。

![Cherry Studio](https://repository-images.githubusercontent.com/805155266/692d7ac1-0787-4b91-99b3-a9231ed6151d)

## 产品概述

Cherry Studio 由 CherryHQ 开发并开源，定位为"AI 生产力工作台"，主要特性包括：

- **多模型统一接入**：一个界面连接 OpenAI、Claude、Gemini、本地 Ollama 模型等主流 LLM 服务
- **300+ 预置 AI 助手**：涵盖写作、编程、翻译、数据分析等多种垂直场景
- **自主智能体（Autonomous Agents）**：支持多步骤任务的自动化执行
- **智能对话管理**：支持对话分支、历史管理等高级功能

该项目在 GitHub 上持续获得关注，近期登上全球趋势榜，反映出用户对多模型统一管理工具的强烈需求。

## 背景与意义

随着 AI 大模型服务商日益增多，用户面临"AI 工具碎片化"的困境——不同任务需要在不同平台之间切换。Cherry Studio 等工具的崛起，正是在填补这一市场空白：提供单一入口统一管理多个 AI 服务。

类似产品还有 LM Studio（主打本地模型管理）、Open WebUI（面向 Ollama 的 Web 界面）等，但 Cherry Studio 在桌面端体验和助手生态的丰富程度上有其独特定位。

开源策略也使得社区可以自由贡献助手模板和功能扩展，降低了二次开发门槛。
