# 五角大楼曾将Anthropic列入黑名单，数小时后OpenAI签署相似协议

据Reddit用户整理的一手资料，五角大楼此前曾因Anthropic拒绝移除监控保护条款而将其列入黑名单；而数小时后，OpenAI与国防部签署的合作协议中，反而保留了相同的保护条款。

![Pentagon Anthropic OpenAI](https://plus.unsplash.com/premium_photo-1681487637194-5ff8e1b82143?crop=entropy&cs=tinysrgb&fit=max&fm=jpg&w=1080)

**事件经过**

Reddit r/Anthropic 社区一位用户发帖，整理并引用了多份一手文件，梳理出以下时间线：Anthropic因坚持保留涉及监控美国公民的安全保护条款，被五角大楼排除在合作名单之外；随后OpenAI与国防部达成协议，而该协议文本中实际上包含了与Anthropic主张相近的保护条款。帖子在社区引发广泛关注。

**背景与意义**

这一对比揭示了AI公司与军方合作谈判中的复杂博弈。Anthropic因坚守安全红线付出了被排斥的代价，而OpenAI最终签署的协议版本却包含了相近条款，由此引发外界对两家公司实际立场差异的重新审视。此事也是当前AI军事应用争议中值得关注的细节之一。
