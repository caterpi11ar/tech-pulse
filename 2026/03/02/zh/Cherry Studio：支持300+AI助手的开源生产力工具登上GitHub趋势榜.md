# Cherry Studio：支持300+AI助手的开源生产力工具登上GitHub趋势榜

![Cherry Studio](https://repository-images.githubusercontent.com/805155266/692d7ac1-0787-4b91-99b8-3d7e9afcce50)

开源AI生产力工具Cherry Studio近日登上GitHub全球趋势榜，该项目支持接入300余个AI助手，提供统一的多模型访问界面。

**事件经过**

Cherry Studio由CherryHQ团队开发，集智能对话、自主Agent和多模型统一接入于一体，支持包括主流闭源和开源大语言模型在内的300+助手配置。项目在GitHub趋势榜获得11分，引发开发者社区关注。

**背景与意义**

随着可用的大语言模型数量持续增长，用户在不同模型和平台之间切换的需求也随之上升。Cherry Studio提供了一个统一的桌面端入口，类似于此前的AI聚合工具，但强调对Agent工作流的原生支持。其开源属性意味着用户可以在本地部署，数据不必经过第三方服务器。
