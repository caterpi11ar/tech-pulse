# Claude推出记忆导入功能，用户可无缝从ChatGPT迁移

Anthropic近日上线新功能，允许用户将ChatGPT等其他AI平台的聊天记录直接导入Claude，保留对话上下文与个人记忆。

![Claude Import Chat](https://cdn.prod.website-files.com/6889473510b50328dbb70ae6/68c469d185a7f0f9b1234567-claude-import.jpg)

**事件经过**

功能入口位于 claude.com/import-memory，用户可将来自ChatGPT、Gemini等平台的历史对话数据上传，Claude将自动解析并建立对应的记忆档案。该功能随即在Reddit多个社区引发大量讨论，获得超过17分的Hacker News关注度。

**背景与意义**

跨平台迁移成本一直是用户切换AI工具的主要障碍——在某个平台积累的上下文、偏好设置和对话历史很难带走。此功能直接解决了这一摩擦点，时间节点与OpenAI军事合作争议引发的大规模用户流失高度重合。同期，Claude应用在Apple App Store多个国家地区跃升至免费榜首位，超过ChatGPT。
