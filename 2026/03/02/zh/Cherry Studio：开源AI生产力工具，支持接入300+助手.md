# Cherry Studio：开源AI生产力工具，支持接入300+助手

开源 AI 生产力工具 Cherry Studio 登上 GitHub 趋势榜，项目支持统一接入300余个 AI 助手，集智能对话、自主 Agent 与多模型管理于一体。

![Cherry Studio](https://repository-images.githubusercontent.com/805155266/692d7ac1-0787-4b91-99b8-3d7e9afcce50)

**事件经过**

Cherry Studio 由 CherryHQ 团队开发，提供桌面端统一入口，支持包括 Claude、GPT-4、Gemini 及各类开源模型在内的300余个助手配置，并原生支持 Agent 工作流。项目在 GitHub 趋势榜获得11分，引发开发者社区关注。

**背景与意义**

随着可用的大语言模型数量持续增长，管理多个 AI 工具的复杂度也随之上升。Cherry Studio 提供了统一的桌面端管理界面，其开源属性意味着数据可在本地处理，无需经过第三方服务器。相较于此前的 AI 聚合工具，该项目对 Agent 工作流的原生支持是一个值得关注的差异化特性。
