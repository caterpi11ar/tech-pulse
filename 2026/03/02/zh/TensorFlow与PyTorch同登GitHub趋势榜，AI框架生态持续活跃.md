# TensorFlow与PyTorch同登GitHub趋势榜，AI框架生态持续活跃

![TensorFlow GitHub](https://opengraph.githubassets.com/a0e2e43724fb7dca217e4304bfe4afb39c738ceaf8961b7adbb8e1e0e8b20cfb/tensorflow/tensorflow)

TensorFlow和PyTorch近期同时出现在GitHub全球趋势仓库列表中，显示AI基础框架的开发者关注度持续保持高位。

**事件经过**

GitHub趋势榜数据显示，tensorflow/tensorflow和pytorch/pytorch两大开源深度学习框架均进入近日的全球趋势排行。此外，Ultralytics YOLO、Keras、Streamlit、Gradio、DeepSpeed、Ray、PyTorch Lightning等AI/ML相关项目也同期上榜。

**背景与意义**

此次多个AI基础框架和工具同期上榜，反映了当前开发者对AI/ML基础设施的持续关注。TensorFlow由Google主导，PyTorch由Meta主导，两者长期是深度学习领域的主流框架。YOLO、Keras等工具的上榜则体现了计算机视觉和高层API需求的增长。Streamlit和Gradio的出现表明AI应用快速原型开发工具同样备受关注。
